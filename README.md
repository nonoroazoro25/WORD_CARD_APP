# 英语单词卡片应用

基于 PyQt5 开发的桌面单词学习应用，支持间隔重复算法（SM-2），帮助高效记忆单词。

## 功能特性

- 📚 **单词卡片**：可翻转的卡片界面，正面显示单词，背面显示释义
- 🔄 **间隔重复**：基于 SM-2 算法，根据记忆情况自动安排复习时间
- 📊 **学习统计**：显示总单词数、新单词、待复习、已掌握等统计信息
- 💾 **数据持久化**：自动保存学习进度，关闭应用后数据不丢失
- 📥 **导入单词**：支持从文本文件或 JSON 文件批量导入单词
- 🎯 **记忆评估**：四个等级（忘记/困难/良好/简单）评估记忆情况

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python main.py
```

## 使用说明

### 添加单词

1. 点击"➕ 添加单词"按钮
2. 输入单词和释义
3. 单词会自动添加到单词库

### 学习单词

1. 在单词列表中点击要学习的单词
2. 点击"🔄 翻转"按钮查看释义
3. 根据记忆情况点击评价按钮：
   - ❌ **忘记**：需要重新学习
   - 😓 **困难**：记忆困难
   - ✅ **良好**：记忆良好
   - 🎉 **简单**：记忆简单

### 导入单词

支持两种格式：

**文本格式** (`words.txt`)：
```
apple|苹果
banana|香蕉
hello|你好
```

**JSON 格式** (`words.json`)：
```json
{
  "words": [
    {"word": "apple", "meaning": "苹果"},
    {"word": "banana", "meaning": "香蕉"}
  ]
}
```

## 间隔重复算法

应用使用 SM-2 算法（SuperMemo 2）来安排复习时间：

- **忘记/困难**：重置间隔，降低难度系数
- **良好**：增加间隔时间，提高难度系数
- **简单**：大幅增加间隔时间，提高难度系数

连续多次良好/简单且间隔超过 30 天的单词会被标记为"已掌握"。

## 数据存储

学习数据保存在 `word_data.json` 文件中，包含：
- 单词列表
- 学习进度
- 复习记录
- 下次复习时间

## 界面说明

- **左侧面板**：单词列表和管理操作
- **中间面板**：单词卡片和学习操作
- **右侧面板**：学习统计和记录

## 快捷键

- `◀ 上一个`：切换到上一个单词
- `下一个 ▶`：切换到下一个单词
- `🔄 翻转`：翻转卡片查看释义

## 注意事项

- 建议每天坚持学习，效果更佳
- 定期复习待复习的单词（红色标记）
- 已掌握的单词（绿色标记）可以偶尔复习巩固
