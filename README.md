# 英语单词卡片应用

基于 PyQt5 开发的桌面单词学习应用，支持间隔重复算法，帮助高效记忆单词。

## 功能特性

- 📚 **单词卡片**：可翻转的卡片界面，正面显示单词，点击卡片查看释义
- 🔄 **间隔重复**：根据记忆情况自动安排复习时间，优化学习效率
- 📊 **学习统计**：饼图显示新单词、待复习、已掌握比例，中心显示已掌握百分比
- 💾 **数据持久化**：使用 SQLite 数据库自动保存学习进度
- 📥 **导入单词**：支持从文本文件（TXT）或 CSV 文件批量导入单词
- ✏️ **编辑单词**：双击单词列表项即可编辑单词和释义
- 🎯 **记忆评估**：两个等级（忘记/掌握）评估记忆情况

## 安装依赖

```bash
pip install PyQt5
```

## 运行应用

```bash
python main.py
```

## 使用说明

### 添加单词

1. 点击左侧面板的"➕ 添加单词"按钮
2. 输入单词和释义
3. 点击"确定"保存

### 学习单词

1. 在左侧单词列表中点击要学习的单词
2. **点击卡片**查看释义（卡片会自动翻转）
3. 根据记忆情况点击评价按钮：
   - ❌ **忘记**：需要重新学习，重置复习间隔
   - ✅ **掌握**：记忆良好，增加复习间隔

### 编辑单词

- **双击**单词列表中的任意单词即可编辑

### 导入单词

支持两种格式：

**文本格式** (`words.txt`)：
```
apple|苹果
banana|香蕉
hello|你好
```

**CSV 格式** (`words.csv`)：
```csv
word,meaning
apple,苹果
banana,香蕉
hello,你好
```

## 间隔重复算法

应用使用简化的间隔重复算法来安排复习时间：

- **忘记**：重置间隔为 1 天，降低难度系数
- **掌握**：根据复习次数增加间隔（1天 → 3天 → 7天 → 递增），提高难度系数
- 当间隔 ≥ 30 天且复习次数 ≥ 5 次时，单词会被标记为"已掌握"

## 数据存储

- **数据库位置**：`~/Library/Application Support/单词卡片/word_card.db`（macOS）
- **数据内容**：
  - 单词列表（单词、释义）
  - 学习进度（复习次数、间隔、难度系数）
  - 复习记录（评价、时间）
  - 下次复习时间

应用会自动从旧版 JSON 文件迁移数据到数据库（如果检测到）。

## 界面说明

- **左侧面板**：单词列表和管理操作（添加、导入、删除、清空）
- **中间面板**：单词卡片和学习操作（上一个/下一个、忘记/掌握按钮）
- **右侧面板**：学习统计（总单词数、饼图、已掌握数量）

## 操作说明

- **切换单词**：点击"◀ 上一个"或"下一个 ▶"按钮
- **查看释义**：点击卡片即可翻转查看
- **编辑单词**：双击单词列表中的单词
- **删除单词**：选中单词后点击"🗑️ 删除单词"

## 注意事项

- 建议每天坚持学习，效果更佳
- 定期复习待复习的单词
- 已掌握的单词会显示在饼图的绿色区域
- 数据自动保存，无需手动保存
